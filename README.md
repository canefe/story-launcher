A Rust + React based Tauri (multi-platform desktop) app used to manage modpacks and instances for StoryMC server. It simplifies the process of downloading and managing modpacks for the server, without the need for the user to manually download and manage them.

![Story Launcher](https://i.imgur.com/2N1DiWL.png)

It allows:
- Downloading mods from Modrinth
- Downloading extra mods from external sources
- Using manifests and file hash system to determine the version of the modpack and mods

Currently supports MultiMC-based launchers such as MultiMC and PrismLauncher.
